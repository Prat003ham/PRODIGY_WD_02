# ⏱️ Stopwatch Web Application
A simple and responsive **Stopwatch Web App** built using **HTML, CSS, and JavaScript**. This project allows users to measure time with **Start, Stop, and Reset** functionalities.

# 🚀 Features
* ▶️ Start Timer
* ⏸️ Stop/Pause Timer
* 🔄 Reset Timer
* ⏱️ Accurate time tracking (hours : minutes : seconds)
* 📱 Fully Responsive Design
* 🎨 Clean and user-friendly interface

# 🛠️ Technologies Used
* HTML5 – Structure of the application
* CSS3 – Styling and layout
* JavaScript – Timer logic and functionality

# 🎮 How It Works
1. Click Start ▶️ to begin the stopwatch
2. Click Stop ⏸️ to pause the timer
3. Click Reset 🔄 to reset the time to `00:00:00`
